## Introduction

HellRat is a HELLDIVERS™ 2 mod install and manage tool. It is built with `Rust` and `Tauri2`, and the frontend is built with `Vue3` and `Element Plus`.

## Features
- Add mods from local files and rename them as needed.
- Pick grid or list view, whatever you like.
- Drag and drop mods to rearrange the load order.
- Text Input: Use Ctrl+Space to bring up the input box, then type text (such as Chinese) and press Enter (no need to activate the in-game input box).
- Quick Phrases: Use Ctrl + . to pull up the panel, then click on a text area.

## Documentation

To check out docs, visit [hellrat.luckyriko.com](https://hellrat.luckyriko.com/).

## Install

A Windows version is already provided. For Linux, users are welcome to build the software from source on their own.

Windows: You can install it from [Releases](https://github.com/luckyriko/hellrat/releases) and [123684.com](https://www.123912.com/s/gZDRTd-aZKKd?pwd=6666#) is also recommended.

## Upgrade

Just download the latest version, reinstall.

## Preparation

You need to install [node](https://nodejs.org/) and [git](https://git-scm.com/) locally. The project is based on [tauri2](https://v2.tauri.app//), [vue3](https://vuejs.org/), [pinia](https://pinia.vuejs.org/), [vue-router](https://router.vuejs.org/), [vue-i18n](https://vue-i18n.intlify.dev/), [vite](https://vite.dev/), [sortable](https://github.com/sortablejs/Sortable) and [element-plus](https://element-plus.org/).
Understanding and learning this knowledge in advance will greatly help the use of this project.


## Dev

```bash
git clone https://github.com/luckyriko/hellrat.git
cd hellrat
pnpm install
pnpm run rat
```

## Build ( Windows10+ )

```bash
pnpm run publish
```
